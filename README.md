Prerequisites

Java JDK 8 or higher

Apache Tomcat (v8 or above)

MySQL Database

Steps

Clone the repository:

git clone https://github.com/avishjhalani/EMS.git


Import the project into your IDE as a Dynamic Web Project.

Configure Apache Tomcat Server in the IDE.

Create the database using scripts available in the sql folder.

Update database credentials in the DAO/config file if required.

Run the project on the server and access it via browser:

http://localhost:8080/EMS/
